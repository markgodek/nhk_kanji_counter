📦 Two Containerized Databases to Use
1. Raw Data DB: MongoDB (NoSQL)
Why MongoDB?

Flexible schema: Ideal for storing unstructured scraped data (e.g., HTML, JSON).

Great for rapid inserts of web-scraped documents.

Handles deeply nested data (e.g., news page sections).

Good for showcasing NoSQL knowledge.

🧬 Suggested DAG Structure
Create a DAG with the following tasks:

with DAG("nhk_kanji_pipeline", ...) as dag:

    scrape_task = PythonOperator(...)
    store_raw_task = PythonOperator(...)
    transform_task = PythonOperator(...)
    load_counts_task = PythonOperator(...)
    notify_or_visualize_task = PythonOperator(...)  # optional
